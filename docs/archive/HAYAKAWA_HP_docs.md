## 2. HPの位置づけと目的

### 2.1 HPの役割
このHPは**売り込みのLPではない**。以下の役割を持つ中継地点として設計する。
- Instagram、GBP（Googleビジネスプロフィール）と連携する情報の母体
- 各種SNS・外部サービスへのハブ（中継地点）
- お店に興味を持った人が雰囲気を感じる場所

### 2.2 流入経路の想定
- **想定する流入**: Instagram、GBP、口コミなど（すでに興味を持った人）
- **想定しない流入**: オーガニック検索（SEO目的ではない）

### 2.3 この位置づけから導かれる設計方針
- 情報を詰め込まない
- ビジュアル・雰囲気を前面に
- 詳細情報（営業時間詳細、地図など）はGBPに任せてリンクで誘導
- 売り込み感・煽り感を出さない

## 3. お店のコンセプトとトーン

### 3.1 お店の特徴

**ハード面**
- 外装：すっきりしたモダンなビジュアル
- 壁：モルタル
- カウンター：天然木
- 照明：ランプシェード（天井吊り下げ）

- 薄皮でパリッと焼き上げ
- 定番3種：ニラにんにく、ショウガ、シソ
- 月替わりの限定餃子あり

### 3.2 「こうではない」の定義（重要なニュアンス）

以下のイメージは**避ける**：
- ガツガツ系餃子屋
- サッと食べてパッと出る店
- 一般的な餃子屋のイメージ（赤提灯、ビール＆餃子、大衆感）
- 激ウマ！ボリューム満点！的な煽り表現
- 赤・黄色などの派手な配色

### 3.3 目指すイメージ
- 落ち着いている
- すっきり、ミニマル
- コミュニケーション重視（会話を楽しむ場）
- 女性的な柔らかさ
- カフェのような空気感
- 「餃子屋っぽくない餃子屋」

### 3.4 来てほしい人
- コミュニケーションしたい人
- ゆったり過ごしたい人
- 女性一人でも安心して入りたい人

## 5. ページ構成

### 5.1 シングルページ構成
縦スクロールの1ページ構成：

#### セクション1: ファーストビュー（外装）
| 項目 | 内容 |
|------|------|
| ビジュアル | 外装（お店のフロント） |
| 要素 | ロゴ（HAYAKAWA）+「一口餃子専門店」 |
| 任意要素 | 短いタグライン（検討中） |
| トーン | すっきり、ミニマル、期待感 |

**設計意図**:
- 外装をFVに使う理由：すっきりした見た目で世界観を確立
- 「餃子屋であることが一瞬で伝わりにくい」は問題なし（オーガニック検索想定していない）
- 「食欲を刺激するビジュアルではない」も問題なし（ガツガツ系ではないため）
- ただしテキストで「一口餃子専門店」は入れる（餃子屋であることは伝えたい）

#### セクション2: 店内の空気感
| 項目 | 内容 |
|------|------|
| ビジュアル | 店内写真 |
| 見せたいもの | モルタル壁、天然木カウンター、ランプシェード |
| コピー例 | 「餃子屋っぽくない、餃子屋。」|
| 枚数 | 1〜2枚程度 |

**設計意図**:
- おしゃれな空間であることを視覚的に伝える
- カウンター席の雰囲気が伝わると良い
- 説明しすぎず、写真で語る

#### セクション3: 今月の餃子
| 項目 | 内容 |
|------|------|
| 内容 | 月替わり限定メニュー |
| 要素 | 写真 + 商品名 + 説明 +「○月限定」ラベル |
| 役割 | 再訪のきっかけ、鮮度感 |

**設計意図**:
- 代表メニューより上に配置
- 理由：再訪者が「今月は何？」をすぐ確認できる
- 「動きがあるサイト」に見える効果

#### セクション4: 代表メニュー
| 項目 | 内容 |
|------|------|
| 内容 | 定番餃子、定食、スープ餃子など3〜4品 |
| 表現 | カルーセル的（横スクロール） |
| トーン | シンプル、押し売り感なし |

**設計意図**:
- 縦に長くならずスマートに複数品を見せる
- パッと見はシンプルだが、調べたい人には情報がある構成

#### セクション5: 店舗情報フッター
| 項目 | 内容 |
|------|------|
| 店名ロゴ | あり |
| 営業時間 | テキストで記載（通常時間のみ） |
| 営業日カレンダー | **画像エリア（Square）** ※必須で用意 |
| 予約ボタン | 外部リンク（Square） |
| Googleマップボタン | 外部リンク |
| 住所 | テキスト |
| 電話番号 | タップで発信可能に |
| SNSリンク | Instagram等 |

**重要**: 営業日カレンダーの画像エリアは必ず確保すること。画像差し替えで更新する運用。

### 5.3 固定バー（常時表示・画面下部）
| ボタン | リンク先 | 種類 |
|--------|----------|------|
| メニュー | ページ内（メニューセクションへ） | ページ内リンク |
| 予約 | Square予約ページ | 外部リンク |
| 店舗情報 | ページ内（フッターへ） | ページ内リンク |
| 地図 | Googleマップ | 外部リンク |

**設計経緯**:
- 当初「今月の餃子」「代表メニュー」を分けていた
- 統合して「メニュー」1枠に、予約を追加
- 理由：固定バーは最重要アクションに絞るべき。予約は明確なゴール（来店につながる）

### 5.4 採用しなかった要素
| 要素 | 理由 |
|------|------|
| バーガーメニュー | シングルページ+固定バーで導線は完結。すっきりした世界観に合う |
| お知らせバー（ぐるぐる回るテキスト） | 読みにくい、落ち着いたトーンに合わない |
| SNS埋め込みセクション | お知らせ程度なら営業日カレンダー画像で対応可能。優先度低 |

#### 選定したペルソナ

**メインペルソナ：沙織さん（28歳・看護師）**
- 基本属性：
  - 名前：沙織
  - 年齢：28歳
  - 性別：女性
  - 職業：看護師
  - 居住地：飯田市内
- ライフスタイル：
  - 普段の過ごし方：シフト勤務の合間にリフレッシュ
  - 食に対する姿勢：友人と美味しいものを食べに行くのが好き
  - SNS利用状況：Instagram・LINEがメイン
- HAYAKAWAとの接点：
  - どこで知ったか：友人のストーリーズで発見
  - なぜHPを見に来たか：雰囲気確認、友達と行けるか
  - HPで確認したいこと：店内写真、予約方法、営業日

  - 想定する来店シーン：平日夜のシフト終わり、友人と
  - お店に期待すること：おしゃれ空間で美味しい餃子
  - 来店の決め手になりそうなこと：SNS映え、女性が入りやすそう
- 選定理由：
  - 味も雰囲気も重視するので、両方を訴求できる
  - 友人と来店するパターン → 口コミ・拡散効果が期待できる
  - 「友人を巻き込んで来店」という波及力がある

**サブペルソナ：美咲さん（32歳・市役所勤務）**
- 基本属性：

  - 性別：女性
  - 職業：市役所勤務（事務職）
